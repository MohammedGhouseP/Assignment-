CSS Box Model Demo

Box 1: box-sizing: content-box
This box uses content-box (default). The width only includes the content,
padding and border add to the final size.
Box 2: box-sizing: border-box
This box uses border-box. The width includes content, padding, and border. Notice how different it looks!
Box 3: box-sizing: content-box (default)
A box with custom padding, border, margin, and a smaller width, demonstrating the additive effect.
